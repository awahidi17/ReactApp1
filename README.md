# Focus Tracker App

A simple React + TypeScript application for tracking focus sessions.

## Features

- Start a focus session by entering a task name
- View a live timer that counts upward in seconds
- Stop the active session at any time
- Automatically log completed sessions
- Review completed sessions with task name and duration
- View quick summary stats for completed work

## Concepts Used

- Functional components and JSX
- `useState` for state management
- `useEffect` for timer lifecycle logic
- Event handling with `onClick`, `onSubmit`, and `onChange`
- Rendering lists from state
- TypeScript interfaces and typed props
